# 🌟 Day 4 – Flow Builder (Salesforce)

---

## 🚀 1. What is Flow Builder?

**Flow Builder** is a powerful automation tool in Salesforce that helps us **automate business processes without writing code**.

👉 It allows you to:

* Create workflows visually (like a flowchart)
* Automate repetitive tasks
* Guide users through screens
* Update records automatically

💡 **In simple words:**
Flow Builder = *Drag-and-drop tool to automate work in Salesforce*

---

## 🔄 2. Types of Flows

Salesforce provides different types of flows depending on the use case.

### 🖥️ Screen Flow

* This flow **interacts with users**
* It shows screens (forms) where users can enter data
* Used for step-by-step guided processes

📌 **Example:**

* Employee onboarding form
* Customer feedback form

💡 Think of it like: *a mini application inside Salesforce*

---

### ⚡ Record-Triggered Flow

* Runs **automatically when a record is created, updated, or deleted**
* No user interaction needed
* Works in the background

📌 **Example:**

* When a new Lead is created → send email notification
* When Opportunity is closed → update related records

💡 Think of it like: *auto-reaction system*

## 💡 3. My Automation Ideas (5 Examples)

### 1️⃣ Auto Welcome Email

### 2️⃣ Lead Assignment

### 3️⃣ Task Reminder System

### 4️⃣ Opportunity Update

When Opportunity stage becomes “Closed Won” → update Account status to “Active Customer”.
