1. *📘 WHAT IS CRM?*

## 🔹 CRM = Customer Relationship Management

CRM is a technology used by companies to manage:

* 👥 Customers
* 📞 Communication
* 💰 Sales
* 🤝 Relationships

It helps businesses keep all customer information in one organized system.

## 🧠 Easy Understanding

Imagine a company storing:

* Customer names
* Phone numbers
* Emails
* Purchases
* Meetings

All in one smart digital platform 👉 That is CRM.





2. *☁️ WHAT IS SALESFORCE?*

Salesforce is a **cloud-based CRM platform** used worldwide.

It helps companies:

* 📈 Increase sales
* 🤝 Improve customer relationships
* 📊 Analyze business data
* 🔒 Store data securely


# 🚀 Why Companies Use Salesforce

| 🔹 Feature           | ✅ Benefit              |
| -------------------- | ---------------------- |
| Cloud Storage        | Access data anywhere   |
| Automation           | Saves time             |
| Reports & Dashboards | Better decision making |
| Customer Tracking    | Better communication   |
| Security             | Safe customer data     |





3.  *IMPORTANT SALESFORCE TERMS:*

A) 🏢 *Account*

An **Account** represents a company or business.

### 📌 Example:

* Mondocorp
* Amazon
* Infosys

### 🧠 Memory Tip:

> Account = Company


B) 👤 *Contact*

A **Contact** is a person associated with an Account.

### 📌 Example:

| Account   | Contact      |
| --------- | ------------ |
| Mondocorp | Shawn Corbin |

### 🧠 Memory Tip:

> Contact = Person


C) 💼 *Opportunity*

An **Opportunity** represents a potential business deal or sale.

### 📌 Example:

Selling a replacement gas generator to Mondocorp.

### 🧠 Memory Tip:

> Opportunity = Deal
